三步安装

开发者模式加载，1 分钟搞定

1

下载源码

从 GitHub 下载 zip 并解压到本地文件夹。

前往下载 →
2

打开扩展管理页

在 Edge 地址栏输入 edge://extensions/，开启左下角"开发人员模式"。

3

加载扩展

点击"加载已解压的扩展程序"，选择解压后的 NJU-Hub 文件夹，固定到工具栏即可。
