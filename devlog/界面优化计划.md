# NJU Hub 界面优化计划

> 从 MD3 Motion System 动效体系出发，逐步统一插件全部 UI 至 Material Design 3 设计语言。

---

## 一、动效放缓（2026-06-16）

### 背景
原有加载动画使用 MD3 令牌但节奏偏快，需要更舒缓的入场体验。

### 改动

#### Popup (`popup/style.css`)
| 动画 | 旧 | 新 |
|------|-----|-----|
| Header 揭示 `popupReveal` | 300ms, 8px, scale 0.98 | **500ms, 12px, scale 0.97** |
| 卡片入场 `popupCardIn` | 400ms, 16px, 延迟 50/120/190ms | **600ms, 20px, 延迟 80/180/280ms** |

#### Options (`options/options.css`)
| 动画 | 旧 | 新 |
|------|-----|-----|
| 页面揭示 `akariMotionReveal` | 400ms, 12px | **600ms, 20px, scale 0.98** |
| Section 切换 `sectionIn` | 400ms, 8px | **500ms, 16px** |
| Card 交错入场 `cardEntrance` | 400ms, 24px, 延迟 0~240ms | **600ms, 32px, 延迟 0~400ms** |
| Schedule Item 列表 `scheduleItemIn` | 400ms, 12px, 延迟 0~280ms | **500ms, 16px, 延迟 0~420ms** |

## 二、侧边栏导航按钮重设计（2026-06-16）

### 设计规范
- **结构**：`[22px SVG Icon] + [14px/500 文字]`，flex row，gap 12px，padding 12px 16px，border-radius 999px
- **图标**：Material Icons Round 内联 SVG，`translateY(1.5px)` 视觉居中
- **8 个导航项分 4 区**（`nav-divider` 间隔 24px，区内 gap 4px）

### 三态

| 状态 | 浅色 | 暗色 | 动效 |
|------|------|------|------|
| Default | 字 `#44474F`，透明底 | 字 `#9CA0AC` | — |
| Hover | 底 `#E8E8EF`，字 `#1A1B21`，图标 scale(1.1) | 底 `#2E3038`，字 `#E2E4EB` | 250ms standard |
| Active | 底 `#D3E4FD`，字 `#001C3B`，weight 600，shadow | 底 `#1E3148`，字 `#D0E2FF` | 图标 scale(1.05) |
| Press | scale(0.97) | 同 | 120ms spring `(0.34,1.56,0.64,1)` |

### 过渡分轨
- `bg/color/shadow` → 250ms standard
- `transform` → 120ms spring

### 涟漪动效
- 导航项独立动画：`opacity 0.2→0`，`scale 0→1`，500ms，完成后移除 DOM
- 其他按钮保持原有 `scale 0→4` 扩散涟漪

### 改动文件
- `options/options.html` — 添加 SVG 图标 + `nav-divider` 分区
- `options/options.css` — 重写 `.nav-item` / `.nav-icon` / `.nav-menu` / `.nav-divider`，新增 `navRippleIn` keyframe，定义 `--nav-*` 颜色令牌 + 暗色覆盖
- `options/options.js` — `attachRipple()` 支持 `animClass` 参数，nav-item 传入 `'animate-nav'`

## 三、Emoji → SVG Icon 替换（2026-06-16）

### 原则
- 消除 content script 中所有功能 emoji（💭⚠️🎉📥），替换为 Material Icons 内联 SVG
- 统一 `fill="currentColor"` 风格，与 options 导航图标一致
- 不同步 `single_js_fyp/`（独立参考文件）

### 图标映射

| 旧 Emoji | 新 Material Icon | 用途 |
|----------|-----------------|------|
| 💭 思考气泡 | `progress_activity` 圆弧 + spin 动画 | eval 按钮「正在评价...」 |
| ⚠️ 警告 | `play_arrow` 三角箭头 | eval 按钮「开始自动评教」 |
| 🎉 庆祝 | 去 emoji（alert 不支持 HTML） | 评教完成提示 |
| 📥 收件箱 | `download` 箭头+托盘 | LMS 课件下载浮动球 |

### 改动文件
- `scripts/eval.js`
  - 新增 `ICONS` 常量（start/spinner/check 三个 SVG 字符串）
  - 注入 `@keyframes nju-eval-spin` 样式（`#nju-eval-style` 防重复）
  - `updateButtonUI()` — `innerHTML` 从 emoji 改为 `ICONS.start` / `ICONS.spinner`
  - 完成提示去 emoji
- `scripts/lms.js`
  - 新增 `DL_ICON` 常量为 download SVG
  - `renderDownloadBall()` — emoji 替换为 `${this.DL_ICON}`
  - `fetchResources()` — `b.innerText = '📥'` → `b.innerHTML = this.DL_ICON`

## 四、MD3 Outlined 自定义下拉框（2026-06-16）

### 设计规范
- **Trigger**：56px 高，8px 圆角，`#74777F` 1px 边框，左侧选中文字 + 右侧 `arrow_drop_down` 24px SVG
- **三态**：Default → Hover (`#1A1B21`) → Open (`#4A90D9` 2px, padding 补偿)
- **箭头**：Open 时 `rotate(180°)`，200ms standard
- **菜单**：绝对定位 trigger 下方 4px，白底 12px 圆角，`0 4px 20px rgba(0,0,0,0.15)`，入场 `translateY(-4px)→0` + opacity
- **选项**：Hover `#E8E8EF` 120ms / Selected `#4A90D9` 字 weight 600 + `#D3E4FD` 底
- **暗色模式**：全部 `--dd-*` 令牌自适应
- **交互**：click-outside 关闭 / keyboard ↑↓EnterEscape

### 替换范围
- `#ui-font-family` — 字体选择（3 个选项）
- `#course-my-campus` — 校区选择（4 个选项）

### 改动文件
- `options/options.html` — 2 个 `<select>` 替换为 `.nju-dropdown` 结构
- `options/options.css` — 移除旧 `select` 样式，新增 `--dd-*` 令牌 + `.nju-dropdown-*` 全部样式 + 暗色覆盖
- `options/options.js`
  - 新增 `NjuDropdown` 类（open/close/toggle/setValue/getValue/onChange/keyboard/click-outside/静态 closeAll/getById）
  - `setVal()` 自动检测 `.nju-dropdown` 并调用 `setValue(val, silent=true)`
  - `applyFont()` 改用 `NjuDropdown.getById('ui-font-family').setValue()`
  - 字体 `onChange` 绑定改为 `fontDropdown.onChange(...)`
  - 保存时 `NJU_CAMPUS` 改为 `NjuDropdown.getById('course-my-campus').getValue()`

## 五、Modal 弹窗 + MD3 输入框

### 动机
- `alert()` / `confirm()` 不可定制样式，阻塞 JS，不同平台渲染不一致
- 原生 `<input>` 无 MD3 缺口边框（fieldset+legend 切口）效果

### Phase 1 — 基础组件库 ✅（2026-06-16 完成）

| # | 产出 | 状态 |
|---|------|------|
| 1 | `libs/nju-modal.css` — 遮罩/面板/关闭按钮 + keyframe 备用 + reduced-motion | ✅ |
| 2 | `libs/nju-modal.js` — `NjuModal.open/close/alert/confirm` + `createMdInput/createMdTextarea` | ✅ |
| 3 | 工厂函数 `createMdInput()` / `createMdTextarea()`（含 HTML 转义） | ✅ |

**API 清单：**
- `NjuModal.open(title, bodyHtml)` — 打开弹窗，返回 overlay 元素
- `NjuModal.close()` — 动画关闭 + 移除 DOM
- `NjuModal.alert(title, message)` — 替代 `alert()`，返回 Promise
- `NjuModal.confirm({title, message, confirmText, cancelText, danger, onConfirm})` — 替代 `confirm()`
- `NjuModal.createMdInput({id, label, type, value, required})` — 返回 MD3 输入框 HTML
- `NjuModal.createMdTextarea({id, label, rows, value})` — 返回 MD3 文本域 HTML

**关键设计：**
- `injectStyles()` 自动注入压缩版 CSS 到 `<head>`，`#nju-modal-style` 防重复
- 无 `onclick` 属性，全部 `addEventListener`（Chrome Extension 兼容）
- 关闭按钮使用 Material Icons close SVG
- input placeholder `" "` 实现 `:not(:placeholder-shown)` 浮动 label 逻辑

### Phase 2 — 替换现有原生弹窗 ✅（2026-06-16 完成）

| # | 目标 | 状态 |
|---|------|------|
| 4 | `scripts/eval.js:193` — `alert()` → `NjuModal.alert('评教完成', ...)` | ✅ |
| 5 | eval.js 依赖 — `manifest.json` 添加 `libs/nju-modal.js` 到 eval content_scripts | ✅ |
| 6 | `options.js` — 5 处 `alert()` 全部替换为 `NjuModal.alert()` | ✅ |
| 7 | `options.js` — 4 处 `confirm()` 全部替换为 `NjuModal.confirm({danger:true,...})` | ✅ |
| — | `options.html` — 加载 `libs/nju-modal.js` | ✅ |
| — | `options.js` 保存/主题逻辑优化 — `_currentSourceColor` 防止色板漂移 | ✅ |

**替换清单：**
- `dm-delete`: `alert('请先勾选...')` → `NjuModal.alert()` / `confirm(...)` → `NjuModal.confirm({danger:true})`
- `dm-clear-all`: `confirm(...)` → `NjuModal.confirm({danger:true})`
- `clear-schedule`: `alert('当前没有课表记录...')` / `confirm(...)` / `alert('课表已清空。')` → NjuModal 全家桶
- `course-db-import`: `alert('导入完成')` / `alert('导入失败...')` → `NjuModal.alert()`
- `course-db-clear`: `confirm(...)` → `NjuModal.confirm({danger:true})`
- `eval.js`: `alert('全部评教任务已顺利清空！')` → `NjuModal.alert('评教完成', ...)`

**关联改动：**
- `manifest.json`: eval content_scripts 添加 `libs/nju-modal.js`
- `options.html`: 添加 `<script src="../libs/nju-modal.js"></script>`

### Phase 3 — 输入框升级（可选）

| # | 目标 |
|---|------|
| 8 | options 页原生 `<input>` / `<textarea>` 升级为 MD3 Outlined 缺口边框 |

### 动效规格
- **入场**：遮罩 `rgba(0,0,0,0→0.32)` 300ms decelerate / 面板 `opacity 0→1 + scale(0.82→1) + translateY(32px→0)` 450ms bounce `(0.18,1.25,0.4,1)`
- **退场**：遮罩 250ms accelerate / 面板 `opacity 1→0 + scale(1→0.92) + translateY(0→12px)` 250ms accelerate，完成移除 DOM
- **关闭按钮**：44px 圆形触摸区，hover `#E8E8EF`，active scale(0.9)

### 适配要点（Chrome Extension）
- `onclick="closeModal()"` 在 content script 隔离世界不可用 → `addEventListener`
- CSS 通过动态 `<style>` 注入，class 加 `nju-` 前缀防冲突
- `showConfirm` 返回 Promise 而非回调
