AI 配置教程

1. 获取 API Key — 前往大模型服务提供商的开放平台注册并获取 API Key。常用提供商包括 SiliconCloud、OpenAI、智谱 AI 等。注册后在控制台创建 API Key 并复制。

2. 配置 API Base URL — 将提供商提供的 API 地址填入对应模块的「API Base URL」输入框。通常格式为 https://api.xxx.com/v1，具体以提供商文档为准。

3. 填写 API Key — 将步骤 1 获取的 Key 粘贴到「API Key」输入框中，该 Key 仅保存在你本地浏览器的插件存储中。

4. 选择模型 — 在「Model」输入框中填写模型名称（如 Qwen/Qwen3-8B）。注意需要支持视觉识别的模型才能用于自动登录验证码识别。

5. 多个模块独立配置 — 自动登录、选课助手、自动评教三个模块的 AI 配置互相独立，你可以为不同模块使用不同的 API Key 和模型。
